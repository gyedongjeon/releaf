# PR: Release v0.1.0

## Summary
Prepares the first official release of **Re:Leaf** (v0.1.0).

## Release Info
-   **Version**: 0.1.0
-   **Type**: Initial Release
-   **Description**: Core features including Ebook Mode, Themes, Settings, and robust content parsing.

-   **Engineering**:
